هذه الالة الحاسبة لديها دوال مختلفة تستخدمها مثلا دالة الجمع التي تأخد عددين ك مدخلات وترجع مجموعهما 


def add(n1, n2):
    return n1 + n2

وبنفس الطريقة مع الطرح والضرب و القسمة 


def subtract(n1, n2):
    return n1 - n2


def multiply(n1, n2):
    return n1 * n2


def divide(n1, n2):
    return n1 / n2

الان سوف نقوم بتخرين هذه العمليات في قاموس بحيث المفاتيح هي العمليات والقيم هي اسماء الدوال التي قمت بكتابتها 


operations = {
    "+": add,
    "-": subtract,
    "*": multiply,
    "/": divide,
}

الان عندما نمرر مفتاح للقاموس مثلا + ونخزن النتيجة في متغير يسمى function اذن النتيجة هي عبارة عن الدالة ونستطيع تمرير عددين لها اذن على حسب ما نمرره للقاموس سوف تعمل function عمل مختلف او عملية مختلفة 


function = operations["+"]
function(5, 10) # The result will be 15

الان سوف نسأل عن عددين 


num1 = int(input("What's the first number?: "))
num2 = int(input("What's the second number?: "))

ثم سوف نسأل ان يتم اختيار واحدة من العمليات التي في القاموس بحيث نقوم بطباعة المفاتيح ثم نسأل ان يختار واحدة من العمليات 


for symbol in operations:
    print(symbol)
operation_symbol = input("Pick an operation from the line above: ")

الان سوف نستعمل رمز العملية الذي اختاره المستخدم ونمرره الى القاموس لكي نستعمل الدالة المناسبة ونرجع العملية الرياضية التي اختارها ونقوم بتمرير العددين ونقوم بطباعة النتيجة 


calculation_function = operations[operation_symbol]
answer = calculation_function(num1, num2)
print(f"{num1} {operation_symbol} {num2} = {answer}")
