تمرين عدد الايام في الشهر

في درس اليوم سوف تجد حل التمرين من الدروس السابقة هو هل السنة كبيسة او لا ؟ 


def is_leap(year):
    if year % 4 == 0:
        if year % 100 == 0:
            if year % 400 == 0:
                print("Leap year.")
            else:
                print("Not leap year.")
        else:
            print("Leap year.")
    else:
        print("Not leap year.")

اولا سوف تقوم بتغيير الدالة is_leap بحيث في مكان طباعة السنة كبيسة او لا سوف ترجع قيمة True او False ثم سوف تقوم بكتابة دالة days_in_month التي سوف تأخذ العام والشهر كمدخلات

حل التمرين:

اولا سوف نقوم بتغيير الدالة is_leap حتى ترجع قيمة True او False في مكان الطباعة هكذا: 


def is_leap(year):
if year % 4 == 0:
    if year % 100 == 0:
        if year % 400 == 0:
            return True
        else:
            return False
    else:
        return True
else:
    return False

والان سوف نستعملها في دالة days_in_month لمعرفة متى نرجع العدد الصحيح من الايام للشهر 


def days_in_month(year, month):
    month_days = [31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31]  
    if is_leap(year) and month == 2:
        return 29
    return month_days[month - 1]

بحيث اذا كانت السنة كبيسة والشهر هو فيفري سوف نرجع 29 و في الحالات الاخرى سوف نرجع قيمة الشهر من القائمة month_days لانها كلها متشابهة للسنه الكبيسة وغير الكبيسة

يمكنك ايضا جعل الدالة ترجع رسالة في حالة ان المستخدم ادخل رقم شهر غير صحيح مثلا 13 هكذا: 


def days_in_month(year, month):
    if month > 12 or month < 1:
        print("Invalid month.")

    if is_leap(year) and month == 2:
        return 29
