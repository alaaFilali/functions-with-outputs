دوال بمخرجات

سوف نبدأ بالتحدث حول الدوال فلقد رأينا الدوال من قبل في الدروس السابقة

النوع الاول من الدوال هي الدوال التي لاتملك مدخلات ولا مخرجات مهمتها فقط تساعدنا على تقليل عدد الاسطر البرمجية في البرنامج الخاص بنا لو اردت تنفيذها بشكل متكرر 


def my_function():
    # Do this
    # Then do this
    # Finally do this

my_function()

النوع الثاني من الدوال هو الدوال بمدخلات بحيث تأخذ مدخلات داخل الاقواس ويتم استعمال تلك المدخلات داخل الدالة التي تم تمريرها عند منادات الدالة وهذا يساعدنا على جعل الدالة الخاصة بنا تفعل اشياء مختلفة بناء على مدخلات مختلفة 


def my_function(something):
    # Do this with something
    # Then do this
    # Finally do this
            
my_function(123)

النوع الاخير هو الدوال بمخرجات رغم انني يمكنني ان اعطيها مدخلات ولكن لم افعل ذلك لان المدخلات والمخرجات منفصلان عن بعضهما البعض 


def my_function():
    result = 3 * 2
    return result

my_function()

كما ترون لقد استعملت كلمة جديدة وهي return والتي تمكنني من ارجاع قيمة من داخل القائمة وما سوف يحدث هو عند منادات الدالة سوف تأخد قيمة result مكان اسم الدالة my_function ويمكنني حفظها في متغير هكذا 


output = my_function()

اي ان قيمة المتغير output بعد منادات الدالة هي العدد 6

لنقم بصنع دوال بمخرجات واول دالة هي دالة تقوم بعمل format للاسماء بحيث سوف تأخد الاسم الاول والاسم الثاني وتحولهم الى title case مهما كانت حالتهم عندما ادخلهم المستخدم

فلو ادخل المستخدم ALAA FILALI مثلا او aLAa FIlaLI النتيجة سوف تكون Alaa Filali بحيث جعلت الحروف الاولى كبيرة فقط وهذا هو title case

فلو قمت ببحث صغير في قوقل لعرفت ان الدالة المستعملة لتحقيق ذلك هي title 


def format_name(f_name, l_name):
    print(f_name.title())
    print(l_name.title())

format_name("alaa", "ALAA")

نتيجة الكود: 


Alaa 
Alaa

فلو تلاحظ قليلا ان دالة title تقوم بارجاع قيمة f_name و l_name في شكل title case اي اننا في مكان ان نقوم بطباعتها يمكن ان نخزنها في متغير 


def format_name(f_name, l_name):
    formated_f_name = f_name.title()
    formated_l_name = l_name.title()
    
    print(f"{formated_f_name} {formated_l_name}")

format_name("aLAA", "alAa")

نتيجة الكود: 


Alaa Alaa

في مكان ان نقوم بطباعة الاسم الجديد يمكننا ان نرجعه بنفس الطريقة كدالة title باستعمال الكلمة return وهكذا الاسم الجديد يصبح مخرج الدالة 


def format_name(f_name, l_name):
    formated_f_name = f_name.title()
    formated_l_name = l_name.title()
    
    return f"{formated_f_name} {formated_l_name}"

formated_string = format_name("aLAA", "alAa")
print(formated_string)

ونفس نتيجة الكود السابق

كان يمكنك ايضا ان تقوم بوضع منادات الدالة داخل ال print بدون تخزينها في متغير وسوف تفعل نفس الشيء 


def format_name(f_name, l_name):
    formated_f_name = f_name.title()
    formated_l_name = l_name.title()
    
    return f"{formated_f_name} {formated_l_name}"

print(format_name("aLAA", "alAa"))

لقد كنا نستعمل هذا النوع من الدوال التي لم نصنعها بانفسنا من بداية الدروس مثل الدالة len بحيث تأخذ string مثلا وترجع طوله 


print(len("Alaa"))

واهم شيء اليوم هو الكلمة return فهي التي تحدد مخرج الدالة واي شيء يأتي بعدها يقوم بأخد مكان منادات الدالة

وفي الدرس القادم سوف نتعمق في هذه الكلمة اكثر واكثر.
