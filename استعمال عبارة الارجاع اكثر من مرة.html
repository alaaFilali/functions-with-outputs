يمكن للدالة ان تكون لديها اكثر من عبارة ارجاع ويمكن ايضا ان تكون لديها عبارة ارجاع فارغة مثلا نريد ان نتحقق ان كان f_name أو l_name فارغين اذن نستعمل عبارة ارجاع فارغة 


def format_name(f_name, l_name):
    if f_name == "" or l_name == "":
        return
    formated_f_name = f_name.title()
    formated_l_name = l_name.title()
    return f"{formated_f_name} {formated_l_name}"
    
print(format_name("aLAA", "alAa"))

فرضا سألنا المستخدم عن اسمه الاول والثاني ولم يدخل شيئا وقمنا بتمريرهما كمدخلات للدالة اذن f_name و l_name فارغين في تلك الحالة لا نريد للدالة الخاصة بنا ان تكمل تنفيذ كل الاسطر لانها بدون فائدة بل نريد تخطي تنفيذ تلك الاسطر باستعمال عبارة ارجاع return فارغة وهكذا سوف نضمن ان تلك الاسطر لن تنفذ الا اذا تم تمرير اسماء وليس فراغ

ببساطة سوف ننهي الدالة مبكرا بهذه الطريقة

ولكن من الافضل الا نرجع فراغ بل نرجع رسالة تخبر المستخدم او المبرمج لما تم انهاء الدالة مبكرا ك "لم تدخل مدخلات صحيحة او ادخلت مدخلات فارغة اعد المحاولة"
